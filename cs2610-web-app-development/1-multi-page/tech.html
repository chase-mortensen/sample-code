Git Command Cheat Sheet

title: This tag affects the text in the title of the site - look at the text in the tab for the example.

h1: This tag is heading one, which is by default the largest heading. It is used for the heading at the top of this page. Example (sans-serif):

This is h1

h2: This tag is heading two, which behaves like heading one, but is by default slightly smaller. Example (sans-serif):

This is h2

p: This is the paragraph tag - it can be used to break text into sections and adds space between elements. This paragraph is an example, and it is also used in each of the definitions in this cheat sheet.

code: This element displays text in monospace, like source code. It is an inline element, which means it doesn't add a space between itself and the last element. Example: This sentece is placed between the opening and closing code tags.

a: This is the hyperlink tag - it can be used to link to external websites. This tag is used with the href attribute, which contains the URL. Example: here

img: This is the image tag. It links to and displays an image on the webpage. It is used with the src attribute. Example:

ol: This is the ordered list tag. It is generally used to create numbered lists. Between the opening and closing ol tags, the elements of the list are created using the following tag.

li:This is the list element tag. It creates entries in either ordered or unordered lists. Example (ol and li tags):
